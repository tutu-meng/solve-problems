# Project Instructions

1. The user will provide a series of problems to solve.
2. You must solve each problem in a reasonable time.
3. The output must be the numeric answer.
4. Save each problem in a separate folder, using the problem number as the name of the folder.
5. Inside each problem folder, create a nested folder named after the AI model that solved it, and place the solution files there.
6. Note that `answer.txt` is added to `.gitignore` and will not be tracked by Git.
7. Inside each problem folder, create a nested folder named "question" to copy the question in html. 
8. add an answer.txt to note the numberic answer
