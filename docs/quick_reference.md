## Quick Reference

Use this repository layout for new Project Euler work:

```text
NNN/
  README.md
  <model-name>/
    solve.py
    solve.cpp
```

Problem-specific notes belong in `NNN/README.md`, including:
- Mathematical derivation.
- Algorithm summary.
- Mermaid diagram when helpful.
- Verification commands and expected sample outputs.
- Final answer.

Recent examples:
- `994/README.md`
- `995/README.md`

Do not add new problem-specific writeups to shared `docs/`.
