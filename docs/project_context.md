## Project Context

This repository is a collection of Project Euler solutions. Each problem should be self-contained in its numbered folder.

Problem folders contain:
- The fetched or restated problem, when useful.
- The derivation and algorithm notes.
- Verification commands and sample checks.
- The final numeric answer.

Shared docs should describe repository conventions only. For current problem writeups, see the relevant problem folder, such as `994/README.md` or `995/README.md`.
